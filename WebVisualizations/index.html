Summary: Latitude vs. X

The intention of this project was to use a single day's weather data, across over 500 cities, to investigate global weather trends. All weather data was collected through OpenWeather API, using a random sampling of cities, all on 1/29/2020.

Matplotlib was used to generate all of the visualizatons contained within this site, and the analyses chosen were meant to represent key aspects of weather patterns (temperature, cloudiness, humidity, and wind speed) relative to the latitude from which the data was collected. Analyses for each trend can be found in the "Visualizations" sidebar, and all source data is included.
